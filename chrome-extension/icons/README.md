# RecallBin Chrome Extension Icons

Since AI image generation is unavailable, create simple icons manually:

## Quick Way: Use Favicon Generator

1. Go to https://favicon.io/favicon-generator/
2. Settings:
   - Text: **R**
   - Background: **Rounded**
   - Background Color: **#8b5cf6** (purple)
   - Font: **Roboto Bold**
   - Font Size: **80**
3. Download and extract
4. Rename and copy to this folder:
   - `android-chrome-192x192.png` → `icon128.png`
   - `android-chrome-512x512.png` → `icon128.png` (resize to 128x128)
   - Create 16x16 and 48x48 versions

## OR: Use Existing Logo

If you have a RecallBin logo file, just resize it to:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

## Temporary Placeholder

For now, you can use any simple 128x128 purple icon file as a placeholder.

The extension will work without icons, they're just for visual branding in the Chrome toolbar.
